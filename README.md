### Tech Stack used
- ExpressJS
- NodeJS
- PeerJS
- socket.io
- EJS
- HTML
- CSS

### Setup locally
- Clone this repo
- Go inside the folder
- Do `npm install`
- Run `nodemon start` or `node server.js`
